# Iris Classification using Naive Bayes

This project builds a machine learning model using the Iris dataset to classify flower species.

## Project Overview

The goal is to predict the species of iris flowers based on features such as sepal length, sepal width, petal length, and petal width.

## Steps

- Data loading
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature selection
- Model training (Naive Bayes)
- Model evaluation

## Dataset

Dataset source: Kaggle - Iris Species Dataset

Dataset link: <https://www.kaggle.com/datasets/uciml/iris>

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
